FAQ*
// frequently asked
What software does this server support? ▾
We primarily focus on plugins for Adobe After Effects and Premiere Pro. More tools may be added in the future based on community needs.
Are these links safe to use? ▾
All links are vetted by the hunnids team before being posted here. That said, always use caution when downloading software and run a virus scan if you're unsure. If a link seems off, let us know in the Discord.
How do I install a plugin? ▾
Check out our Installing Plugins guide in the Guides section for a full walkthrough including installation paths and first-time setup tips.
After Effects crashes on launch. How do I fix it? ▾
Try clearing your AE cache and preferences first — go to Edit → Purge → All Memory & Disk Cache. If it keeps crashing, check that your GPU drivers are up to date and that you're running a compatible version of Windows or macOS. You can also ask for help in the Discord.
The installer is asking for a serial number. What do I do? ▾
Check the folder the installer came in — serial numbers or activation instructions are usually included alongside the download. If you can't find it, reach out in the Discord and someone can help you out.
Where do I go if I need help? ▾
Jump in the Discord! We have dedicated help channels where members and staff can assist with installs, plugins, general editing questions and more. The community is super active so you'll usually get a response fast.
